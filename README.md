## Overview  
Keen Media House is a modern, responsive, one-page website built for a premium photography and videography company. It includes a portfolio gallery, services, pricing, testimonials, about section, and a working contact/booking form.

## Features  
- Fully responsive (mobile, tablet, desktop)  
- Beautiful portfolio gallery with lightbox  
- Services & pricing packages  
- About section  
- Client testimonials carousel  
- Smooth animations & scroll effects  
- Contact form (Formspree / Netlify Forms ready)  
- SEO-friendly + Open Graph tags  
- Fast loading & optimized images  

## Tech Stack  
- HTML5  
- CSS3 (Flexbox + Grid)  
- Vanilla JavaScript (ES6+)  
- Lightbox2  
- Google Fonts  
- No build tools required  

## Deployment (Ready in under 5 minutes)

### Option 1: Netlify (Recommended - Free)
1. Click this button  
[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/Aubrey359/Keen-Media-House)

### Option 2: Vercel
1. Go to https://vercel.com  
2. Import this GitHub repo  
3. Deploy instantly (zero config)

### Option 3: GitHub Pages
1. Go to repo → Settings → Pages  
2. Source: Deploy from branch → main → /(root)  
3. Save → Site live at: https://aubreys359.github.io/Keen-Media-House/

### Option 4: Any Traditional Hosting
- Download the repo as ZIP  
- Upload all files to your public_html / www folder
